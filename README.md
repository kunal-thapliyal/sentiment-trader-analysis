# # 📊 Market Sentiment vs Trader Behavior Analysis

## 📌 Overview
This project analyzes how market sentiment (Fear/Greed) impacts trader behavior and performance on Hyperliquid.

The goal is to identify patterns between sentiment and trading outcomes and derive actionable insights that can help improve trading strategies.

---

## 📊 Datasets

The project uses two datasets:

1. **Bitcoin Market Sentiment (Fear/Greed Index)**
   - Contains daily sentiment classification (Fear, Greed, Neutral)

2. **Historical Trader Data (Hyperliquid)**
   - Includes trade-level data such as price, size, side, PnL, and timestamps

---

## ⚙️ Tech Stack

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  
- Jupyter Notebook  

# Setup & How to run
1. Clone the Repository
2. Install Libraries (pip install pandas numpy matplotlib seaborn scikit-learn jupyter)
3. Launch Jupyter Notebook
